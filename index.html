Sound Explorer

An interactive sound pad to create and mix beats using your keyboard or mouse

Volume control uses the Web Audio API's GainNode to adjust sound levels. The mute button toggles between 0 and the current volume value.
Changes the OscillatorNode type in Web Audio API, affecting the sound's timbre. Each waveform produces a different harmonic content.
Recording uses MediaRecorder API to capture audio output. Playback uses an AudioBufferSourceNode to replay the recorded sequence.

Use your keyboard (A-K) or click the pads to play sounds
