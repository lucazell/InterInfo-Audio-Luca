Sound Explorer

An interactive sound pad to create and mix beats using your keyboard, mouse, or touch

Volume control uses the Web Audio API's GainNode to adjust sound levels. The mute button toggles between 0 and the current volume value.
Changes the OscillatorNode type in Web Audio API, affecting the sound's timbre. Each waveform produces a different harmonic content.
Recording uses MediaRecorder API to capture audio output. Playback uses an AudioBufferSourceNode to replay the recorded sequence.
This visualizer shows real-time audio frequencies using Web Audio API's AnalyserNode. The rainbow-colored waveform represents different frequency ranges - lower frequencies on the left, higher on the right. The height of each bar indicates the volume at that frequency, creating a dynamic visual representation of the sounds you play.

Use your keyboard (A-K), click, or tap the pads to play sounds

Note: If a sound gets stuck, refresh your browser to clear all active sounds

On mobile devices, make sure your device isn't in silent mode and tap a pad to start the audio.

This page was created by Luca Zell during the winter term 2024/25 in the Interactive Information course of the Interface Design program at the University of Applied Sciences Potsdam.

This project is built using the Web Audio API, which provides powerful tools for processing and synthesizing audio in web applications. Features include oscillators for sound generation, gain nodes for volume control, and analyzers for visualizing audio frequencies.
